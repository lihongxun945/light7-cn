light7

light7 is light and easy to use UI lib forked from Framework7。I would like to make it easy to build a webapp UI on both Android 4.0+ and iOS 6+ using Light7.

Very Light

It is only a UI lib, but not a framework. Most time you just copy component's HTML code and paste in your project, and It works!

Light7 depend on [zeptojs](http://zeptojs.com/), nothing to learn if you have used jQuery before.

iOS Specific

Thanks to Framework7, Light7 can realize precisely pixel-perfect iOS native app's interface.

powerful components

Light7 provide many powerful component such as Picker, Calendar and Pull-to-refresh.
