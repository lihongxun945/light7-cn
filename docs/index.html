为什么选择 Light7 ?

轻量且强大

Light7 是一个库而不是一个框架。它提供了 20+ 个常用的组件，却只有 82K JS 和 117K CSS 大小的两个文件。大部分时候你只需把组件的HTML代码复制到自己的HTML文件中即可。

Light7 依赖 jQuery 或者 zeptojs, 如果你以前用过 jQuery/Zepto 那么没有任何上手难度.

炫酷的iOS风格

我们的组件都是按照iOS风格设计的，所有有很多组件都是iOS独有的炫酷设计。

强大的组件

Light7 提供了很多功能强大的组件 - Pull-to-refresh, Action Sheet and Notification.

换肤

我们提供了四种主题色，以及夜间模式，点击查看换肤文档。
